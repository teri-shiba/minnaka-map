# minnaka map（ みんなかマップ ）

みんなかマップは、みんなの出発地点から集まりやすい中間地点と、その周辺の飲食店を提案するアプリです。

- 周りに気を遣ってしまい積極的な提案はできないので、集まりやすい場所を提案したい
